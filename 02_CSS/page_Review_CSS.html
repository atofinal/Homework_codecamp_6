• property และ value คืออะไรแตกต่างกันอย่างไร
• property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
• value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร
• ให้แต่ละกลุ่มออกมาอธิบายเพื่อนๆหน้าชั้นเรียน

คำถาม คำตอบ
property และ value คืออะไรแตกต่างกันอย่างไร Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง

Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , แบบอักษร ฯลฯ
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง 5 กลุ่ม
property for content: color , size , font , text, list-style-type, Height, Width, background

property for border : border, table

property for padding : padding (top right buttom left)

property for margin : margin (top right buttom left)

property for element : display, Height, Width, Position, Float, Align, Flex, Grid
value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร 2 ประเภท
• Absolute lenghts
cm เซนติเมตร
mm มิลลิเมตร
in นิ้ว
px พิกเซล 1/96 นิ้ว
pt พอยท์ 1/72 นิ้ว
pc พิคาล 1/12 นิ้ว

• relative lenghts
em เท่าขนาดฟอนต์ของ element
ex เท่าความสูงของฟอนต์ของ element
ch ความกว้างของตัวหนังสือใน parent element
rem เท่าขนาดฟอนต์ของ root element
vh เปอร์เซ็นต์ความสูงหน้าจอ
vw เปอร์เซ็นต์ความกว้างหน้าจอ
vmin เปอร์เซ็นต์ของ view point ต่ำสุด
vmax เปอร์เซ็นต์ของ view point สูงสุด
% เปอร์เซ็นต์ของ parent element


• value ใน display property มีอะไรบ้าง
• grid และ flex แตกต่างกันอย่างไร
• justify and align แตกต่างกันอย่างไร
• position คืออะไร และ มีอะไรบ้าง
• z-index คืออะไรใช้อย่างไร
• แต่ละกลุ่มออกมานำเสนอหน้าห้อง
• ให้ทำตัวอย่างการใช้ position มาทุก property
• ให้แต่ละกลุ่มทำหน้าตา page ตาม link ด้านล่าง
• แต่ละคนพิมพ์ได้คนละ 10-15 นาที
• แต่ละกลุ่มออกมาพรีเซ็นต์หน้าห้อง
